# Distilled UI

## Must-show HUD elements
- Health/durability
- Shell count
- AP + move phase state
- Nib prompt
- Objective text
- Interact prompt

## UI principles
- Tactical data must be explicit and fast to parse.
- Keep text compact; minimize decorative noise.
- Maintain high contrast for cover, pathing, and hazards.
- Reinforce world contrast (warm surface vs clinical Pale Quarter) without harming clarity.
