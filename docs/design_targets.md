# Design Targets (UI / Feel / Readability)

## UI targets
- Information hierarchy is immediate: health, shells, AP/move state, objective, interact prompt.
- Tactical information is visible, not hidden: cover state, movement range, target lines.
- Prompt language is short and consistent.
- Surface vs Pale Quarter styling contrast is subtle but clear (warm/patched vs sterile/clean).

## Feel targets
- Mood blend: **cartoonish + eerie + dusty + warm + lonely**.
- Lottie feels physically small in every mode (cover value, recoil motion, posture animation).
- Kindly feels heavy and consequential (force + audio + movement impact).
- Nib feels useful first, uncanny second.

## Readability targets
- Silhouettes are recognizable at small scale (hat + small frame + oversized gun).
- Terrain edges, cover, gaps, and hazards are readable in three-quarter camera.
- Enemy intent is telegraphed with simple, consistent cues.
- Puzzle affordances are distinct from background decoration.

## Content quality checks
- No room should devolve to “just shoot everything.”
- Each mandatory room uses at least two core verbs.
- Failure recovery is fast (short reset/checkpoint return).
